# 🍳 Aplikasi Dapur Anita

![Logo](images/logo2.png)

Selamat datang di **Aplikasi Dapur Anita**! Platform digital modern untuk memudahkan pelanggan memesan produk dapur, minuman, dan kebutuhan rumah tangga secara online.

---

## 🚀 Fitur Utama

- 🛒 **Belanja Produk**: Lihat, cari, dan beli produk dapur favorit Anda.
- 🧺 **Keranjang & Checkout**: Tambahkan produk ke keranjang dan lakukan pembayaran dengan mudah.
- 📦 **Lacak Pesanan**: Pantau status pesanan Anda secara real-time.
- 👤 **Profil Pelanggan**: Kelola data diri dan histori pesanan.
- 🖼️ **Upload Bukti Bayar**: Proses pembayaran lebih aman dan transparan.
- 🔒 **Login & Register**: Sistem autentikasi aman untuk pelanggan.

## 🛠️ Teknologi

- **Frontend**: Flutter (Android, iOS, Web)
- **Backend**: Laravel REST API
- **Database**: MySQL

---

## ⚡ Cara Menjalankan

### Backend (Laravel)
```bash
cd Backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

### Frontend (Flutter)
```bash
cd flutter_application_1
flutter pub get
flutter run
```
